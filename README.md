# 📊 System Capacity & Care Load Analytics for Unaccompanied Children

## Overview

This project analyzes healthcare system capacity and care-load trends within the Unaccompanied Alien Children (UAC) Program managed by the U.S. Department of Health and Human Services (HHS).

The solution transforms operational data into actionable insights for monitoring care demand, capacity stress, backlog accumulation, and humanitarian response effectiveness.

---

## Project Objectives

### Primary Objectives

* Quantify daily and cumulative care load
* Analyze intake vs discharge balance
* Identify capacity stress periods

### Secondary Objectives

* Support staffing and shelter planning
* Improve operational visibility
* Enable data-driven policy decisions
